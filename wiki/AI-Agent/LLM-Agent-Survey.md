# Large Language Model Agent: A Survey (北大综述)

> 北京大学关于 LLM Agent 的综合性综述，系统梳理方法论、应用场景和核心挑战。

## 核心内容

### Agent 架构方法论

1. **推理 (Reasoning)** — CoT、Self-consistency、Tree-of-Thought 等推理增强技术
2. **规划 (Planning)** — 任务分解、反思与修正、规划库等方法
3. **工具使用 (Tool Use)** — 外部工具调用扩展 Agent 能力边界
4. **记忆 (Memory)** — 短期和长期记忆机制
5. **行动 (Action)** — 与环境交互执行操作

### 应用场景

- **单 Agent 应用**：编程、写作、数据分析、科学研究
- **多 Agent 协作**：社会模拟、协作任务、辩论
- **人机协作**：Agent 辅助决策、创意生成

### 核心挑战

- 幻觉与可靠性
- 长时任务一致性
- 安全与对齐
- 评估与基准
- 多 Agent 协调效率

## 关键要点

- 180KB 的全面综述，覆盖 LLM Agent 的各个维度
- Agent = LLM + 推理 + 规划 + 记忆 + 工具 + 行动
- 应用从单任务向多 Agent 协作演进
