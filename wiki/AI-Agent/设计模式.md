# 智能体设计模式

> Antonio Gullí 和 Jimmy Song 的系统性智能体设计模式著作，涵盖从提示链到多智能体协作的完整模式体系。

## 核心内容

### 智能体分级

- **Level 0：核心推理引擎** — 基础 LLM 能力
- **Level 1：连接型问题解决者** — 单 Agent + 工具调用
- **Level 2：战略型问题解决者** — 多步推理、规划、自适应
- **Level 3：协作型多智能体系统** — 多 Agent 协作

### 智能体五大未来假设

1. **通才智能体的出现**
2. **深度个性化与主动目标发现**
3. **具身化与物理世界交互**
4. **智能体驱动经济**
5. **目标驱动、变形多智能体系统**

### 核心模式

1. **提示链 (Prompt Chaining)** — 将复杂任务分解为顺序的提示步骤
2. **路由 (Routing)** — 根据输入分类，动态选择处理路径
3. **上下文工程** — 作为提示工程的自然延伸

## 关键要点

- 模式对智能体开发至关重要，提供可复用的解决方案
- 智能体能力从单步推理演进到多 Agent 协作
- 未来趋势：通才化、个性化、具身化、经济化
