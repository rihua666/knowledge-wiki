# OpenAI 企业 AI 下一阶段

> 企业收入占比已超 40%，2026 年底有望与消费者业务持平。AI 从实验阶段进入"做真正的工作"阶段。
> 来源: [OpenAI](https://openai.com/index/next-phase-of-enterprise-ai/) | 2026-04-08

## 核心数据

- **企业收入占比**：>40%，年底目标与消费端持平
- **Codex**：300 万周活用户，年内增长 5 倍+
- **API 处理量**：每分钟 150 亿 token
- **ChatGPT**：9 亿周活用户
- **新客户**：Goldman Sachs、Philips、State Farm
- **增长客户**：Cursor、DoorDash、Thermo Fisher

## 战略双引擎

**1. OpenAI Frontier — 企业智能基础设施层**
- 统一管理全公司的 AI Agent
- 客户：Oracle、State Farm、Uber
- 与 AWS 合作构建 Stateful Runtime Environment（有状态运行时）
- 合作伙伴：McKinsey、BCG、Accenture、Capgemini、Databricks、Snowflake

**2. 统一 AI 超级应用**
- 整合 ChatGPT + Codex + Agent 浏览 + 更多能力
- 一个入口管理所有 AI Agent 任务
- 核心优势：9 亿用户已熟悉 ChatGPT，降低企业部署摩擦

## 关键洞察

> "The world is in a phase of **capability overhang** — AI models can already do far more than most people and enterprises are using them for today."

从"让 AI 帮你做任务"到"管理 AI Agent 团队替你做任务"的转变已经开始。GitHub、Notion、NextDoor 已在构建多 Agent 系统执行端到端工程工作。
