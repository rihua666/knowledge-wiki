# 卡兹克创作 Skill

> 将三年公众号写作方法论蒸馏为可复用的内容创作 Skill，完整开源。
> 来源: 数字生命卡兹克 | 2026-04-07

## 核心要点

- **核心思想**：把自己的写作风格、禁忌词、节奏控制、质量标准全部结构化为 Skill 文件，让 AI 辅助创作时严格遵循个人风格 [原文]
- **Skill 内容包含三层**：风格规则（切入角度、禁用词/标点）、方法论（什么是好内容、节奏控制、技巧手法）、四层自检体系 [原文]
- **最佳使用方式排序**：Claude Chat + Opus 4.6 > Cowork + Opus 4.6 > Claude Code + Opus 4.6 [原文]
- **幻觉抑制策略**：主模型不做事实校验，用 Codex + GPT-5.4 单独检测事实错误 [原文]

## 四层自检体系

1. **硬性规则扫描**：禁用词、禁用标点、结构性套话 [原文]
2. **风格一致性检查**：开头从具体场景切入、长短句交替、口语化、独立成段 [原文]
3. **内容质量检查**：观点有场景支撑、聊天感输出、具体事件连接文化参照 [原文]
4. **活人感终审**：读起来是有见识的普通人在跟你聊，还是 AI 在输出信息 [原文]

## 如何构建自己的创作 Skill

1. 扔几篇代表性文章 + 方法论白皮书给 AI，总结成初始 Skill [原文]
2. 让 AI 按当前 Skill 写一篇文章 [原文]
3. 自己动手改（必须改，不能偷懒）[原文]
4. 把 AI 版本和自己版本同时丢回 AI，分析差异，迭代回 Skill [原文]
5. 重复 2-4 共 3-4 轮（不要太多，会过拟合）[原文]

## AI 辅助创作的边界

- **AI 不能做的**：实测内容、教程内容（必须自己体验）[原文]
- **AI 可以辅助的**：找证据写观点、找类比和比喻、确定角度框架后的扩写 [原文]
- **核心原则**：AI 辅助创作 ≠ AI 替你写完。观点是你的，AI 只是弹药。千万不要丢了自己判断的能力 [原文]
