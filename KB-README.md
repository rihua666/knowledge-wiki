# 知识库快速参考
## 🚀 快速开始
\`\`\`bash
# 进入 workspace
cd ~/.openclaw/workspace
# 添加新素材到 raw/
vim raw/2024-01-15-你的内容.md
# 让我编译知识库
# 对我说: "编译知识库" 或 "更新 wiki"
\`\`\`
## 📁 目录结构
\`\`\`
workspace/
├── raw/          # 原始素材（你手动添加）
│   ├── 2024-01-15-文章-XXX.md
│   └── 2024-01-20-论文-YYY.pdf
│
├── wiki/         # 编译后的知识（我自动维护）
│   ├── index.md
│   ├── RAG/
│   ├── LLM/
│   └── ...
│
├── memory/       # 日常日志
├── MEMORY.md     # 长期记忆
└── KB-README.md  # 本文件
\`\`\`
## 💬 常用指令
- **"编译知识库"** - 我会扫描 raw/ 并更新 wiki/
- **"查找关于 X 的内容"** - 我会搜索知识库
- **"给 X 做个总结"** - 我会整理相关内容
- **"检查知识库健康度"** - 我会查找不一致/缺失数据
## 🎯 核心理念
> **LLM 是编译器，不是搜索引擎**
- 主动维护索引 + 摘要
- 自动分类、反向链接
- 以 Markdown 为中心
- 人类只读，LLM 写入
