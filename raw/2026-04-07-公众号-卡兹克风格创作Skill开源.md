卡兹克把自己做了三年公众号的所有方法论蒸馏成一个内容创作Skill，完整开源。

开源地址：https://github.com/KKKKhazix/khazix-skills（khazix-writer那个Skill）

Skill内容包括：
- 风格规则：什么角度切入是卡兹克风格，什么词和标点绝对不用
- 方法论：什么好内容，节奏控制，技巧手法
- 四层自检体系：硬性规则扫描 → 风格一致性检查 → 内容质量审查 → 活人感终审

最佳使用方式：Claude Chat模式 + Opus 4.6 > Cowork + Opus 4.6 > Claude Code + Opus 4.6。幻觉不抑制，用 Codex + GPT-5.4 检测事实。

如何构建自己的创作Skill：
1. 扔几篇代表性文章 + 方法论白皮书给AI，总结成初始Skill
2. 让AI按当前Skill写一篇文章
3. 自己动手改（必须改）
4. 把AI版本和自己版本同时丢回AI，分析差异，迭代回Skill
5. 重复2-4共3-4轮（不要太多，会过拟合）

四层自检体系：
1. 硬性规则扫描：禁用词、禁用标点、结构性套话
2. 风格一致性检查：开头从具体场景切入、长短句交替、口语化、独立成段
3. 内容质量检查：观点有场景支撑、聊天感输出、具体事件连接文化参照
4. 活人感终审：读起来是有见识的普通人在跟你聊，还是AI在输出信息

AI如何正确辅助创作：
- AI不能做的：实测内容、教程内容（必须自己体验）
- AI可以辅助的：找证据写观点、找类比和比喻、确定角度框架后的扩写
- AI辅助创作 ≠ AI替你写完。观点是你的，AI只是弹药
- 核心原则：千万不要丢了自己判断的能力
