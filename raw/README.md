# Raw 素材库

这里存放所有原始素材，作为知识库的输入源。

## 支持的内容类型

- **文章**: 保存全文或链接 + 摘要
- **论文**: PDF 文件或链接 + 核心观点
- **代码片段**: 保存到 .md 文件中
- **图片**: 保存截图 + 描述
- **笔记**: 任何你想记住的内容

## 文件命名建议

raw/
├── 2024-01-15-论文-LLM-knowledge-base.md
├── 2024-01-20-文章-Andrej-Karpathy-观点.md
├── 2024-02-05-代码-有趣的设计模式.md
└── 2024-02-10-图片-Architecture.png
## 工作流
1. **收集**: 看到有趣的内容 → 保存到 raw/ （附上日期和简短标签）
2. **标记**: 使用 #标签 标记主题，如 #RAG #LLM #Python
3. **等待**: 告诉我"编译知识库"，我会处理 raw/ → wiki/
